Hexo博客搭建01

发表于 2019-11-12 更新于 2019-11-17 分类于 Hexo 阅读次数：

之前都是使用jerkyll作为静态网页的,这次想使用HEXO来搭建个人博客
同时也想好好把自己所学习的知识以及生活的感触写在博客里.好好学习,提升个人价值吧.

本地安装与运行

安装Node.js

去官网下载,并安装. HEXO是基于Node.js的

安装HEXO服务器

npm install hexo-cli -g

hexo init blog

cd blog

npm install

hexo server

安装并启动成功 ctrl+c 退出进程

框架的基本结构&打包后的文件介绍

网站生成命令 hexo g 或者 hexo generate

生成后的public文件夹直接作为静态文件可进行部署

主题安装&推荐

hexo.io/themes/

Installation

cd your-blog

git clone https://github.com/geekplux/hexo-theme-typing themes/typing

  • Typing requires Hexo 2.4 and above.

Enable

Modify theme setting in _config.yml to typing.

Also if your website language is English make sure to explicitly specify it: language: “en”

Update
cd themes/typing
git pull

restart

首先ctrl+c关闭进程
再输入 hexo server 启动

字符类

1) [ ] 字符类 中括号里所有的字符进行挨个匹配,只要有一个匹配成功就返回

2) 使用 ^ 取反, 不匹配括号里的字符就行

3) 范围内的匹配
英文字母
pattern = /[a-z]/;

中文
pattern = /[\u4e00-\u9fa5]/;

其他(前面的要比后面的小)
/[a-zA-Z0-9]/

常用字符类

默认情况下de 别名
/[a-zA-Z0-9_]/ -> /\w/
/[^a-zA-Z0-9_]/ -> /\W/

匹配空格 或者 制表符 或者 Unicode空格制表符

/\S/除了以上符号所有字符

匹配数字
